#####Virtual Piano — это приложение для музыкантов, любителей музыки и тех, кто хочет весело провести время, 
играя на виртуальных клавишах. Оно предоставляет удобный интерфейс и функции для игры, записи и воспроизведения музыкальных композиций.


#### Основные возможности:
- Игра на пианино:
  - Поддержка одиночных нот и аккордов.
  - Воспроизведение звуков при помощи мыши или клавиатуры.
  - Звук продолжается, пока удерживается клавиша.
  
- Запись и воспроизведение:
  - Запись композиций в реальном времени.
  - Сохранение записей в формате `.wav` для дальнейшего использования.
  - Загрузка и воспроизведение ранее записанных мелодий.

---

#### Требования к системе:
- Операционная система: Windows 7 и выше.
- Свободное место на диске: 22 МБ.
- Объем установочного файла: 7 МБ.

---

#### Установка:
1. Загрузите файл установщика (setup.exe).
2. Запустите установочный файл и следуйте инструкциям мастера установки.
3. После завершения установки приложение будет доступно через:
   - Ярлык на рабочем столе.
   - Меню "Пуск".

---

##### Руководство пользователя:

#### 1. Открытие приложения:
   - Используйте ярлык на рабочем столе или найдите приложение через меню "Пуск".

#### 2. Игра на пианино:
   - Основное окно приложения содержит виртуальное пианино:
     - Нажимайте на клавиши пианино мышью или соответствующие клавиши на клавиатуре для игры.
     - Названия нот отображаются на клавишах для вашего удобства.
     - Звук длится до тех пор, пока клавиша удерживается.
   - Поддерживается одновременное воспроизведение нескольких клавиш (аккордов).


#### 2.5 Нажатие на клавиши через клавиатуру
      
_____ DO  RE  MI  FA  SOL  LA  SI___________
      |   |   |   |    |   |    |
      V   V   V   V    V   V    V
      A   S   D   F    G   H    J

#### 3. Запись мелодий:
   - Для записи вашей композиции:
     1. Перейдите в главное меню и выберите пункт "Запись".
     2. Начните игру на пианино. Все действия будут записаны.
     3. Для завершения записи снова выберите "Запись" в меню.
     4. Файл автоматически сохранится в формате .wav.

#### 4. Воспроизведение мелодий:
   - Чтобы воспроизвести ранее записанную мелодию:
     1. В меню выберите пункт "Сыграть мелодию".
     2. Укажите файл `.wav` из списка сохранённых записей или другой .wav файл.
     3. Наслаждайтесь воспроизведением!

---

#### Технические характеристики:
- Размеры файлов:
  - Установщик: 7 МБ.
  - Внутренний инсталлятор: 575 КБ.
  - Приложение после установки: 22 МБ.
  - Сам файл исполняемого приложения: 7 КБ.

- Приложение оптимизировано для работы на устройствах с минимальной загрузкой ресурсов.

---

#### Советы и рекомендации:
- Работа с клавишами: Если вы используете клавиатуру для игры, изучите отображение соответствия клавиш.
- Запись сложных аккордов: Для получения лучшего качества записи удерживайте клавиши плавно.
- Проверка звука: Перед запуском программы убедитесь, что на вашем устройстве настроены динамики или наушники.
